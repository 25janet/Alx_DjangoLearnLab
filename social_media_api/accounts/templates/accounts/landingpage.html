Welcome to the Accounts API

Here are the available endpoints:

  • POST /accounts/register/ Create a new user
  • POST /accounts/login/ Log in and get token
  • GET /accounts/profile/ Get user profile (requires token)
  • POST /accounts/follow/<id>/ Follow a user (requires token)
  • POST /accounts/unfollow/<id>/ Unfollow a user (requires token)
